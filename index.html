JZSlider ver. 0.902.03

Instructions:

Step 1

Download the plugin, unzip it, copy files and include JZSlider© script and stylesheet in your document (you will need to make sure the css and js files are on your server, and adjust the paths in the script and link tag). Make sure you also load the jQuery library.

<!-- Add jQuery library -->
<script type="text/javascript" src="http://code.jquery.com/jquery-latest.min.js"></script>

<!-- Add JZlider© -->
<script type="text/javascript" src="/JZlider/JZlider.js"></script>
<link rel="stylesheet" href="/JZlider/JZlider.css" type="text/css">

Step 2

Attach JZSlider© when the document is loaded.

<div class="jzlider">
    <a class="slide" href="#">
        <img src="slide1.jpg">
    </a>
    <a class="slide" href="#">
        <img src="slide2.jpg">
    </a>
    <a class="slide" href="#">
        <img src="slide2.jpg">
    </a>
</div>

<script type="text/javascript">
    $(function () {
        $('.jzlider').JZlider();
    });
</script>

Options

// Default values
slide_speed: 500, // String, sliding speed.
stay_time: 5000, // String, execute interval.
autoslide: true, // Boolean, auto play.
dire_ctr: false, // Boolean, move to the next or the previous slide.
dot_ctr: false, // Boolean, dot controller.
cycling: true, // Boolean, stop at the last slide when set to false.
mouseover_pause: true, // Boolean, pause on mouse over.
debug: false // Boolean, debug mode.
