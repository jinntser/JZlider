JZlider ver. 0.912.29

Instructions:

Step 1

Download the plugin, unzip it, copy files and include JZlider© script and stylesheet in your document (you will need to make sure the css and js files are on your server, and adjust the paths in the script and link tag). Make sure you also load the jQuery library.

<!-- Add jQuery library -->
<script type="text/javascript" src="http://code.jquery.com/jquery-latest.min.js"></script>

<!-- Add JZlider© -->
<script type="text/javascript" src="/JZlider/JZlider.js"></script>
<link rel="stylesheet" href="/JZlider/JZlider.css" type="text/css">

Step 2

Attach JZlider© when the document is loaded.

<div class="jzlider">
    <a class="slide" href="#">
        <img src="slide1.jpg">
    </a>
    <a class="slide" href="#">
        <img src="slide2.jpg">
    </a>
    <a class="slide" href="#">
        <img src="slide2.jpg">
    </a>
</div>

<script type="text/javascript">
    $(function () {
        $('.jzlider').JZlider();
    });
</script>

Options

// Default values
speed: 500, // String, sliding speed.
stayTime: 5000, // String, execute interval.
autoPlay: true, // Boolean, auto play.
direCtr: false, // Boolean, move to the next or the previous slide.
bullet: false, // Boolean, dot controller.
cycling: true, // Boolean, stop at the last slide when set to false.
hoverPause: true, // Boolean, pause on mouse over.
debug: false // Boolean, debug mode.
